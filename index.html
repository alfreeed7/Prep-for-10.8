d3 Algorithm

Select: it is like the selector in css. It will select the first element that matches in the DOM.
Append: will create a new DOM element and append it to the end of the selection that we have in the first step. For instance, d3.select("body").append("p") will insert a paragraph to the end of the body.
Text: insert the string between the openging tag and the closing tag. In our case, we can insert a string between the p.
Style: it allows us to set the HTML attributes and CSS properties on our selections. For example, the original color of the text in the body is black, we can change it.
